# generateSSH-42SP

Shell script to generate SSH Key automatically

## Getting Started

```
git clone https://github.com/caio-vinicius/generateSSH-42SP
```
```
cd generateSSH-42SP
```
```
bash generateSSH.sh
```

### Prerequisites

Linux based-system
